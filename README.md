# Re:Eat SEO and favicon package

Upload every file in this folder to the root of the `ReEat-Landing` repository.

The included `index.html` is your supplied page with only the existing metadata/favicon block in `<head>` replaced. Everything from the existing `<style>` tag onward is unchanged.

Files included:
- `index.html`
- `favicon.ico`
- `favicon-16x16.png`
- `favicon-32x32.png`
- `favicon-48x48.png`
- `apple-touch-icon.png`
- `favicon-192x192.png`
- `favicon-512x512.png`
- `site.webmanifest`
- `og-image.png`
- `reeat-logo.png`
- `robots.txt`
- `sitemap.xml`

After uploading, commit to `main`. GitHub Pages will redeploy automatically.
